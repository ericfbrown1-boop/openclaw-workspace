# Salesforce Account Analytics

## When to Use
- Pipeline review or board prep
- Competitive deal analysis
- Weekly/monthly sales reporting

## Capabilities (Planned)
- Open opportunities by stage and amount
- Win/loss analysis vs Rubrik, Commvault, Veeam
- Pipeline value trending (weekly, monthly, quarterly)
- Key account health monitoring
- Forecast accuracy tracking

## Prerequisites
- Salesforce OAuth credentials (via 1Password Jarvis account)
- API access to Cohesity Salesforce instance
- Connected app setup in Salesforce admin
