# Financial Report Generator

## When to Use

- After earnings-analyzer completes extraction for any company
- Eric requests a formatted competitive comparison report
- Quarterly board prep requiring peer benchmarking
- Weekly competitive summary roll-up
- Any analysis output that needs professional Word doc formatting

## Output Formats

- **Word (.docx)**: Primary format (per USER.md preference)
- **PDF**: Convert from Word via `docx2pdf` or LibreOffice CLI when requested
- **Google Sheets**: Optional for tabular data updates

## Template Sections

1. **Executive Summary** — 1 paragraph, key takeaways
2. **Revenue & ARR Analysis** — Table + YoY/QoQ commentary
3. **Margin Trends** — Gross, operating, net (GAAP and non-GAAP)
4. **Cash Flow Analysis** — OCF, FCF, FCF margin
5. **Valuation Multiples** — Peer comparison table
6. **Product & Use-Case Highlights** — Relevant context from cohesity-domain
7. **Risks & Considerations** — Macro, competitive, execution

## Workflow

1. **Receive data**: Dict/JSON from earnings-analyzer with extracted metrics
2. **Load template**: Base styles and formatting from `skills/financial-report-gen/template.docx` (or create from scratch)
3. **Populate sections**: Fill tables, charts placeholder, commentary
4. **Apply formatting**: Cohesity brand colors (blue #0063B2), consistent fonts
5. **Save to output**: `~/Documents/reports/{company}_{quarter}_{date}.docx`
6. **Email delivery**: Attach and send via `gog gmail send`

## Sample Code: Word Document Generation

```python
from docx import Document
from docx.shared import Inches, Pt, RGBColor
from docx.enum.table import WD_TABLE_ALIGNMENT
from datetime import datetime

def create_earnings_report(data: dict, output_path: str) -> str:
    doc = Document()

    # -- Styles --
    style = doc.styles["Heading 1"]
    style.font.color.rgb = RGBColor(0, 99, 178)  # Cohesity blue
    style.font.size = Pt(16)

    # -- Title --
    doc.add_heading(f"Competitive Earnings Analysis — {data['quarter']}", level=0)
    doc.add_paragraph(f"Prepared for Eric Brown | {datetime.now().strftime('%B %d, %Y')}")
    doc.add_paragraph(f"Generated by Jarvis")

    # -- Executive Summary --
    doc.add_heading("Executive Summary", level=1)
    doc.add_paragraph(data.get("exec_summary", "Summary pending."))

    # -- Revenue Table --
    doc.add_heading("Revenue & ARR Comparison", level=1)
    add_metrics_table(doc, data["revenue_table"])

    # -- Margins --
    doc.add_heading("Margin Trends", level=1)
    add_metrics_table(doc, data["margin_table"])

    # -- Cash Flow --
    doc.add_heading("Cash Flow Analysis", level=1)
    add_metrics_table(doc, data["cashflow_table"])

    # -- Valuation --
    doc.add_heading("Valuation Multiples", level=1)
    add_metrics_table(doc, data["valuation_table"])

    # -- Risks --
    doc.add_heading("Risks & Considerations", level=1)
    for risk in data.get("risks", []):
        doc.add_paragraph(risk, style="List Bullet")

    # -- Footer --
    doc.add_paragraph("")
    footer = doc.add_paragraph("Sent by Jarvis — AI assistant to Eric Brown")
    footer.runs[0].font.size = Pt(8)
    footer.runs[0].font.color.rgb = RGBColor(128, 128, 128)

    doc.save(output_path)
    return output_path
```

## Sample Code: Table Builder

```python
from docx.shared import Pt, RGBColor, Inches

def add_metrics_table(doc, table_data: list[list[str]]):
    """
    table_data: list of rows, first row is header.
    Example:
    [
        ["Metric", "RBRK Q3'25", "RBRK Q2'25", "CVLT Q3'25", "CVLT Q2'25"],
        ["Revenue ($M)", "225.4", "201.1", "248.3", "232.7"],
        ["YoY Growth", "35.2%", "37.8%", "18.4%", "16.9%"],
        ["ARR ($M)", "1,012", "943", "892", "845"],
    ]
    """
    if not table_data:
        return

    rows, cols = len(table_data), len(table_data[0])
    table = doc.add_table(rows=rows, cols=cols)
    table.style = "Light Grid Accent 1"
    table.alignment = WD_TABLE_ALIGNMENT.CENTER

    # Header row
    for j, val in enumerate(table_data[0]):
        cell = table.rows[0].cells[j]
        cell.text = str(val)
        for paragraph in cell.paragraphs:
            for run in paragraph.runs:
                run.font.bold = True
                run.font.size = Pt(9)

    # Data rows
    for i in range(1, rows):
        for j in range(cols):
            cell = table.rows[i].cells[j]
            cell.text = str(table_data[i][j])
            for paragraph in cell.paragraphs:
                for run in paragraph.runs:
                    run.font.size = Pt(9)

    doc.add_paragraph("")  # spacing after table
```

## Sample Code: Email Delivery

```python
import subprocess
import json

def send_report_email(report_path: str, subject: str, body_html: str):
    """Send report via gog gmail send."""
    to_addrs = "ericfbrown1@gmail.com,Eric.brown@cohesity.com"
    cc_addr = "ericfbrown1@gmail.com"

    # Using gog CLI for Gmail send
    cmd = [
        "gog", "gmail", "send",
        "--to", to_addrs,
        "--cc", cc_addr,
        "--subject", subject,
        "--body", body_html,
        "--attach", report_path,
    ]
    result = subprocess.run(cmd, capture_output=True, text=True, timeout=30)
    if result.returncode != 0:
        raise RuntimeError(f"Email send failed: {result.stderr}")
    return result.stdout
```

## Error Handling

- **python-docx not installed**: `pip install python-docx` in venv. Log to incidents.
- **Template missing**: Generate from scratch (code above does not require template file).
- **Email send fails**: Retry once. If gog fails, fall back to Zapier MCP `gmail_send`. Log to `memory/incidents.jsonl`.
- **File path issues**: Default output dir `~/Documents/reports/`. Create if missing.
- **Large tables**: Cap at 8 columns to fit page width. Split into multiple tables if needed.
