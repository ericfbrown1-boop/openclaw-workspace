# Workday HR Analytics

## When to Use
- Board prep or budget reviews
- Headcount planning
- Department cost analysis
- Org structure questions

## Capabilities (Planned)
- Headcount by department and location
- Budget vs actual by cost center
- Payroll cost trending (monthly, quarterly)
- Open requisitions and time-to-fill
- Org structure visualization

## Prerequisites
- Workday API access (via Cohesity IT)
- Read-only integration system user
- Report-as-a-Service (RaaS) endpoints configured
