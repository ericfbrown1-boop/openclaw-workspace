Notes & Known Configuration Details
• PyTorch uses nightly builds (cu128) because the RTX 5080 Blackwell GPU (sm_120) is not yet supported in stable PyTorch releases.
• Claude Code runs from both Windows PowerShell (npm install) and WSL2 (native binary). Prefer WSL2 for projects inside Linux filesystem.
• The 'code .' command from WSL2 opens VS Code on Windows and auto-attaches to the Dev Container via the .devcontainer/devcontainer.json file.
• SSH keys for GitHub are stored inside the Docker container and will not persist if the container is rebuilt. Re-run ssh-keygen and add to GitHub if this occurs.
• Docker Desktop must be running before launching WSL2 terminals or VS Code Dev Containers.
• CUDA Toolkit is NOT installed directly on Windows — all GPU code runs inside Docker containers using the NVIDIA Container Toolkit passthrough.
• PowerShell execution policy is set to RemoteSigned at CurrentUser scope.
